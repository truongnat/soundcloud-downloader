- endpoint: /api/soundcloud/search
- cache = true
- params:
    - q: string; optional
    - litmit: number: optional; default = 10
    - client_id: string

- API playlist: 

- endpoint: /api/soundcloud/playlist
- cache = true
    - url: string; ( url contain client_id in query params)

- API song:

- endpoint: /api/soundcloud/song
- cache = true
    - url: string ( url contain client_id in query params)

- endpoint: /api/soundcloud/get-client-id

    - params:
